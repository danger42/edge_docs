You can take advantage of the Google Analytics service to generate detailed statistics about traffic to your Developer Services portal. To activate the service, you need to enable Google Analytics in the developer portal and then add the Google Analytics web-property ID to the portal's system configuration.

To enable Google Analytics

  1. Log in to your portal as a user with admin or content creation privileges.
  2. Select the Modules entry in the Drupal administration menu.
  3. Select the Statistics section to expand it.
  4. Check the Google Analytics checkbox.

To add a web-property ID

  1. Sign up for the Google Analytics service if you haven't already done so. In response, you'll receive Google Analytics tracking code and a web-property ID. The web-property ID takes the form UA-XXXXX-YY, for example, UA-10876-11.
  2. Select Configuration > System > Google Analytics in the Drupal administration menu. This opens the Google Analytics page.
  3. Add the web-property ID after "UA-" in the Web Property ID field.

  4. Click Save configuration.
