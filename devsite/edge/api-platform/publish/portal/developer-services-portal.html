Note: This topic applies to cloud customers only. If you're an Edge for Private Cloud customer, you'll install and configure your own portal locally. For information on where to download the product installers, see the Apigee Release Notes.

Prerequisites

You must have an Apigee Edge account, as described in Creating an Apigee Edge account.

For the number and types of Developer Services Portal environments available to you, see Apigee Edge Pricing & Features.

Create a portal instantly in the New Edge experience UI

If you're a cloud customer, you can create a lightweight developer portal right away in the New Edge experience UI. Navigate to https://apigee.com/edge, sign-in, and select Publish > Portals in the side navigation bar of the New Edge experience UI. For complete details, see Building your developer portal (New Edge).

Create a Drupal-based portal

Drupal is the original Edge developer portal framework. It provides a rich set of functionality and all the CMS capabilities of Drupal with additional Apigee-developed Drupal modules. However, implementing a Drupal-based portal requires more time and effort.

Current support for portal development using Drupal is available by request for paid accounts only. For more information, see Creating a developer portal. Contact Apigee Support for assistance if you wish to use Drupal for your portal development.
