Following are supported software versions for Apigee cloud and Apigee Edge for Private Cloud (on-premises).

Apigee Edge Cloud

The following are supported in Apigee Edge cloud.

Java (JDK)

  • Oracle JDK 7/8
  • OpenJDK 7/8

Java Callouts in API proxy flows are available only in Apigee Edge Enterprise accounts. See also the Edge pricing features page. 

TLS

Apigee Edge supports TLS version 1.0, 1.1, and 1.2.

Node.js (Trireme)

Trireme 0.8.9 (which supports Node.js 0.10.32 and Rhino JavaScript engine 1.7.7.1). See also Understanding Edge support for Node.js modules.

JavaScript

Rhino JavaScript engine 1.7.7.1

Apigee Edge Microgateway

Apigee Edge Microgateway is supported on all standard platforms that are supported by Node.js. For more information, see the Node.js download page. We support Node.js LTS versions after 4.5 on Windows, macOS, and Linux.

Please note that Edge Microgateway is not supported on those platforms listed under Additional Platforms on the Node.js download page.
