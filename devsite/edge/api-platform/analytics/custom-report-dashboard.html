What does a custom report tell me?

A custom report is a way to specify precisely what you want to measure across your API program. For instance, you can measure all API traffic generated from a specific client IP address. For information on creating custom reports, see Manage custom reports.

A custom report is similar in look and feel to analytics dashboards, however the data that's displayed depends on the metrics and drill-down dimensions you selected when you created the custom report.
