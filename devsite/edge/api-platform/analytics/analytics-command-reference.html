Note: This document provides a list a preconfigured URIs for specific applications of Apigee Analytics Services. This is not a comprehensive reference. For a list of all metrics, dimensions, and filters, see Analytics metrics, dimensions, and filters reference.

For email:password, substitute the username and password for your Apigee account at https://enterprise.apigee.com/login.

For {myorg}, substitute the name of your Apigee organization, which you can find under Account Settings on https://enterprise.apigee.com.

(If you do not have an Apigee account, you can register for one at https://accounts.apigee.com/accounts/sign_up.)

The requests below assume the environment is 'prod'. If you require analytics on another environment (such as 'test'), modify /environments/prod to /environments/test

For more information on the API call used in these examples, see Get metrics organized by dimensions.

API

$ curl -u email:password https://api.enterprise.apigee.com/v1/o/{myorg}/environments/prod/stats/apis?select=sum(message_count)&timeRange=9/24/2012%2000:00~10/25/2012%2000:00&timeUnit=day
